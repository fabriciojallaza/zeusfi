  - View yield pools from backend (with error/empty states)
  - View positions from backend
  - View rebalance history from backend (with error/empty states)
  - Read ENS preferences from backend cache
  - Write ENS preferences on-chain (Ethereum Mainnet via wagmi setText)

  - Analysis overlay (animates AND fetches real LI.FI quote from backend)
